Execution Roadmap

Ruva Catalog Generator

From competitor link to premium catalog images — automated, intelligent, consistent.

The Core Idea

Problem: No tool exists to take a competitor's product listing and automatically generate matching catalog images for your own product.

Insight: Generate a strong Hero Image first, then use it as a visual anchor alongside each competitor image's intent to recreate a full, visually consistent catalog.

Result: A complete set of catalog images that feel like they came from the same photoshoot — matching top Amazon listing quality.

System Flow

1. Competitor Input

Paste Amazon/Flipkart URL or manually upload competitor images + description

2. Your Product

Upload product images or enter a text description

3. Deep Analysis

Gemini analyzes competitor catalog — extracts intent behind each image, identifies info gaps

4. Smart Questions

System asks targeted questions to fill gaps — skip any question for intelligent defaults

5. Hero Image

Generate a strong hero product shot — the visual anchor for all catalog images

6. Full Catalog

Hero image + each competitor intent → generate complete, consistent catalog set

The Key Function

  • generate_catalog_image(hero, competitor_intent, prompt) — THE core approach
  • Sends hero image (visual anchor) + competitor's intent image (style reference) + text prompt to gemini-3-pro-image-preview (Nano Banana Pro)
  • "This is MY product (image 1). Recreate the style/intent of image 2 for my product."
  • Intent-specific prompt additions: lifestyle, closeup, infographic, size comparison, etc.

Consistency

  • Sequential generation — each prompt includes shared style notes
  • Post-generation validation via gemini-3.1-flash-image-preview (consistency score check)
  • All images locked to same aspect ratio

Core Principles

Visual Consistency

All images feel like the same photoshoot. The hero image is the anchor that binds them.

Intent Replication

Each competitor image has a purpose. That purpose is recreated, not the image itself.

Never Block

Missing info? Use intelligent defaults. Scraping failed? Manual upload. Always proceed.

Premium Output

Results should feel like top 0.1% Amazon listings. Clean, conversion-focused, professional.
